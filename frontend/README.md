# Frontend

This is the React + Vite frontend for ResumeAI.

## What you need
- Node.js 18+
- npm (or your preferred package manager)

## Run it locally
1. Open a terminal in the `frontend` folder.
2. Install dependencies:
   - `npm install`
3. Start the dev server:
   - `npm run dev`

Vite will print the local URL (usually `http://localhost:5173`).

## Notes
If the API is running on a different host/port, update the frontend config accordingly.
