# ResumeAI

An open-source AI-powered resume builder. Still in development.

## About

ResumeAI is a lightweight project that combines a Python backend with a modern React frontend. It uses local AI models (Ollama) for processing, so everything runs on your machine with no cloud dependency.

## Tech Stack

- **Backend**: FastAPI with Python
- **Frontend**: React + Vite with SCSS
- **AI/LLM**: Ollama with llama3 (or any Ollama model)
- **Database**: SQLAlchemy + SQLite
- **Styling**: Clean, minimal theme system

## What's Inside

- FastAPI REST API for backend logic
- Pydantic for data validation
- React components with a professional UI
- Local LLM integration via Ollama
- Light and dark theme support
