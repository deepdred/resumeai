# ResumeAI

An open-source AI-powered resume builder that runs locally on your machine using Ollama LLM models. It consists of a FastAPI backend and a React + Vite frontend.

## Tech Stack

- **Backend**: FastAPI with Python
- **Frontend**: React + Vite with SCSS
- **AI/LLM**: Ollama with llama3 (or any Ollama model)
- **Database**: SQLAlchemy + SQLite

## What's Inside

- A Resume Builder which uses Local ollama LLM models like llama3 to generate resume content based on user input.
- A simple React frontend to interact with the backend API.
